Projects
Below are examples of website application projects I developed for various departments during my studies and internship at my college institution.
1. Career Advancement Navigator
Career Advancement Navigator is a robust web application designed for the DepEd Human Resource Department. It simplifies tracking and monitoring personnel development by enabling efficient progress monitoring. Administrators can swiftly implement training programs and select suitable participants using advanced filtering criteria. The application also features an OTP submission for attendance, enhancing security and accountability.
2. Tangb City Booking Sytem
The Tangub City Booking System is a web application designed for residents and visitors to book facilities, rooms, or equipment within Tangub City. Users can create accounts and make bookings at any time, provided the items are available. Administrators have the ability to manage the resources and generate reports based on specified date ranges.
3. Deans List Application System
The Dean's List Application System is a web application for Tangub City Global College. It allows students from different programs within the institution to apply for the dean's list award smoothly and efficiently without the need for physical interaction.
4. LRC Book Accession System
The LRC Book Accession System is a web application for the Learning Resource Center (LRC) at Tangub City Global College. It enables the storage and organization of book data, allowing for quick book accession or retrieval and easy report generation.
5. TCGC Clinic Inventory System
The Clinic Inventory System is a web application designed to store and organize medicine data. It enables quick medicine accession or retrieval and facilitates easy report generation.
6. CISO Logbook System
The CISO Logbook System streamlines the log booking process in the College Information System Office at Tangub City Global College. It transforms the traditional paper-based process into an online system, enabling faster and more efficient transactions within the office.
